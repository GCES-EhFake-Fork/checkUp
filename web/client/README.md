# Como rodar o projeto localmente

## Pré-requisitos
- **Node.js** (v20+) e **pnpm** (ou npm) instalados para desenvolvimento frontend

## 1. Rodando o Frontend (React + Vite)

### Ou rodando localmente
```sh
cd web/client
npm install -g pnpm@latest # se não estiver instalado
pnpm install  # ou npm install
pnpm run dev  # ou npm run dev
```

O frontend estará disponível em http://localhost:5173

## 2. Acessando o sistema
- Frontend: http://localhost:5173
